================================================================================
INSTALLATION INSTRUCTIONS
================================================================================

OPTION 1: M5Burner (Easiest)
-----------------------------
1. Download M5Burner from https://m5stack.com/pages/download
2. Connect M5StickC Plus2 via USB
3. Add custom firmware or select from list
4. Flash LucidWatch-v2.1-FINAL-MERGED.bin
5. Done!

OPTION 2: esptool.py (Advanced)
--------------------------------
esptool.py --chip esp32 --port /dev/ttyACM0 write_flash 0x0 \
  LucidWatch-v2.1-FINAL-MERGED.bin

OPTION 3: PlatformIO (Developers)
----------------------------------
1. Clone repository
2. Open in VS Code with PlatformIO
3. Run: pio run --target upload
